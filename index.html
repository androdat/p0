What is a Custom Rom ?

ROM or read-only memory refers to a phone's firmware based on Google Android Platform. Android is an open-source project therefore any developer can edit the code, re-compile it and create their own versions adding more features and functionality. So a custom ROM is a modified form of the original software developed by the Android Community and released for free.

Although installing custom roms do void your warrenty. In order to do so you need to first unlock your boot loader (how to do it depends on the brand of the device). Then you need to install a custom recovery like TWRP and using that you will be able to install a custom rom
